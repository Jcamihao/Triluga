# Release Notes v2.0.0 - Triluga Redesign

Data: `2026-04-27`

## Resumo

A versão 2.0 consolida o redesign visual do Triluga no frontend Angular, cobrindo as principais telas de navegação, descoberta, autenticação, perfil, anúncios, comparação e mensagens. A entrega também prepara a base de estilos e documentação para evolução das próximas telas.

## Entregas principais

- home redesenhada com busca rápida, atalhos e categorias
- busca e filtros atualizados para a nova linguagem visual
- detalhe do veículo redesenhado com galeria, dados do anúncio e ações principais
- páginas de login, cadastro, favoritos, comparação, chat, perfil e perfil público redesenhadas
- painel do proprietário e fluxo de anúncio redesenhados
- shell da aplicação revisado com menu, navegação inferior, compare tray e estado compartilhado de UI
- card de veículo, modal de filtros e componentes compartilhados alinhados ao redesign
- Tailwind, PostCSS, Material Symbols e scripts de formatação adicionados à base do frontend
- remoção da página legacy de host em favor do fluxo atual de `anunciar-carro`

## Impacto esperado

- experiência mais consistente entre mobile e desktop
- páginas principais mais próximas das telas aprovadas pelo design
- navegação e ações recorrentes mais previsíveis
- base visual mais simples de manter nas próximas ondas do redesign

## Validação

- `npm --prefix frontend run build`

## Observações

- A pasta `RedesignTriluga/` e o arquivo `RedesignTriluga.zip` são materiais de referência local e não fazem parte da release.
