# Release Notes v2.3.9 - Triluga

Data: `2026-05-05`

## Resumo

A versão 2.3.9 fecha ajustes de navegação mobile, ativa o fluxo de salvar alertas de busca no frontend e remove pontos de interação que pareciam funcionais mas ainda não tinham backend disponível.

## Entregas principais

- menu mobile acionável nas telas de home, busca, chat e painel do proprietário
- central de notificações mobile aberta pelo ícone do topo, com lista, estado vazio e ação de marcar como lida
- fechamento automático de menu e notificações ao trocar de rota
- criação de alerta de busca a partir dos filtros atuais na tela de busca
- novo serviço Angular para consumir `POST /search-alerts`, `GET /search-alerts/my` e `DELETE /search-alerts/:id`
- busca rápida da home usando o parâmetro `q`, alinhado com a página de resultados
- feedback para recuperação de senha e login social ainda indisponíveis
- links legais apontando para a política pública enquanto os termos dedicados não existem
- spec do cadastro ajustada para prover `HttpClient`

## Validação

- `npm --prefix frontend run build`

## Observações

- O frontend compila com avisos de budget CSS já existentes em componentes acima do limite configurado.
