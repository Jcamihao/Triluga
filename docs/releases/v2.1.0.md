# Release Notes v2.1.0 - Triluga Marketplace

Data: `2026-04-27`

## Resumo

A versão 2.1 evolui a base do Triluga após o redesign 2.0, reforçando fluxos de backend, serviços Angular, navegação mobile, privacidade e preparação de deploy.

## Entregas principais

- auth, refresh de sessão e recuperação no frontend revisados
- chat, favoritos, notificações, comparação e busca ajustados para estados mais previsíveis
- APIs de admin, analytics, perfis, privacidade, imagens e veículos refinadas
- storage e cache/queue com tratamento mais defensivo
- home, perfil, perfil público, privacidade, favoritos, comparação e filtros refinados visualmente
- runtime config, Tailwind, TypeScript e Angular config alinhados ao build atual
- Docker Compose, Render e docs de QA/release atualizados

## Impacto esperado

- menor chance de falhas silenciosas em sessões e chamadas autenticadas
- jornadas principais mais consistentes em mobile
- deploy mais previsível em ambiente gerenciado
- documentação operacional mais alinhada ao estado atual do produto
