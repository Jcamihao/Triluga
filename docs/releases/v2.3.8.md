# Release Notes v2.3.8 - Triluga

Data: `2026-04-30`

## Resumo

A versão 2.3.8 consolida os ajustes web mais recentes do Triluga, com foco em autenticação, cadastro, navegação padrão e refinamentos das telas desktop sem alterar o fluxo mobile.

## Entregas principais

- página web de login usando header e footer padrão
- checkbox "Lembrar de mim por 30 dias" funcional, com sessão persistente e refresh token de 30 dias
- cadastro web v2 aplicado com layout desktop separado do mobile
- campos de cadastro padronizados entre mobile e web: foto, nome, e-mail, CPF, telefone, nascimento, CEP, cidade, endereço, complemento, UF, senha e confirmação
- validações visuais no cadastro web para e-mail, CPF, telefone, CEP, senha forte e confirmação de senha
- header web padrão reutilizado na página de cadastro
- ajustes visuais no cadastro web, incluindo remoção de prova social e título lateral com o gradiente padrão

## Validação

- `npm --prefix frontend run build`

## Observações

- O frontend compila com avisos de budget CSS já existentes em componentes acima do limite configurado.
